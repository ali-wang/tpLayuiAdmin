Tp6Layui
===============

> 运行环境要求PHP7.1+。


## 主要功能

* 采用Thinkphp6.0开发
* 权限管理
* 目录管理
* 前段基于layuiAdmin
* 统一和精简大量用法


## 文档

[Thinkphp6.0完全开发手册](https://www.kancloud.cn/manual/thinkphp6_0/content)

[layui开发手册](https://www.layui.com)
